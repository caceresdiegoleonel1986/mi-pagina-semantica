# Comandos básicos de Git

- **Repositorio**: carpeta controlada por Git - la versión final guardada
- **Commit**: foto del proyecto en un momento
-**Staging area**: "Zona de preparación" - es la carpeta donde seleccionas lo que quieres publicar
-**Brach**: Línea de tiempo alternativa
-**Merge**: Unir lineas de tiempo
-**Working directory**: Donde editar los archivos (tu escritorio)
